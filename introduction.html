Introduction

"The only way to learn a new programming language
is by writing programs in it."
Brian W. Kernighan
Dennis M. Ritchie
The C Programming Language

Aside:

Python program files end in .py.

Recall that a program is a file that a computer executes. To write a program, then, is to create a file that contains instructions for the computer to execute. Python being an interpretted language, the programs we will be writing will be plain text. That is, they will be readable by you, as well as executable by the computer. It is the role of the Python interpretter to take these files, interpret them, and carry out their instructions.

Getting Started

The latest major version of Python is Python 3, and that is what we will be covering. If you'd like to try it without installing anything, check out Online Python.

To install Python in a Windows environment, the recommended route is to go through the Microsoft Store. If you are installing Python on your machine, a proper text editor (like Microsoft's Visual Studio Code) is also recommended. Note that Microsoft Word is not a text editor, and cannot be used for programming. Other popular text editors include Atom and Notepad++.

In a Linux environment, Python 3 should be already be installed.

On any platform, you can confirm your Python installation by opening a terminal (PowerShell in Windows) with,

$ python3 --version

Hello World

Since the 1970's, the first program in any language has been very simple: Print the message Hello World!. In Python, the program looks like this:

print("Hello World!")
hello.py

This program is your first major hurtle, because in order to be successful, an entire development environment must be in place. If you're having trouble, contact a friend— or get on the forums!

Exercises

  2. Experiment with leaving parts of the program out. What kind of errors do you get?
  3. What does print() do without any arguments? Hint: It does something!

Variables

A variable is a name for an object. To declare a variable is to introduce it into the source file; to bind a variable is to set it to refer to some object. In Python, variables are declared and bound in a single step, with the = operator. To create a variable named msg referring to the string "Hello World!", for example, we would do,

msg = "Hello World!"

Thus, our second program will carry out the same task as hello.py, but incorporates a variable to do so:

msg = "Hello World!" print(msg)
hello2.py

Notice that the variable msg (and not the string "msg") is passed to print().

Python automatically resolves variables to the objects they refer to. A variable can only be bound to a single object at a time, and it is an error to use a variable that has not been declared. In Python, all variables can be rebound at will,[1] meaning that you can always redefine a variable to refer to something else.

Exercises

  1. Try passing "msg" to print(). What do you think the result would be?
  2. Try printing msg, rebinding it, then printing it again.

Getting Input

Alongside print(), Python has a built-in function to retrieve input from the user. The function is input(). It takes at most one argument, a message to be printed, and it returns a string. We can bind the returned value of input() to a variable, then use it later. This is how we "save" input to use it later:

name = input("What is your name? ") print("Hello", name)
salutations.py

Note that the print() function is passed two arguments, the string "Hello" and the variable name. When more than one argument is passed to a function, they are separated by a comma, ,.

The typical structure of a program is to obtain some input from the user, do some work on it, then return some results. Input can be raw, as above, or it could be from a file. Likewise, output can be merely printed, or it can be sent to a file.

Exercises

  1. Try passing a variable bound to a string to input(). What do you think the result would be?
  2. Try passing more than one argument to input() What do you think the result will be?

Numbers of Various Sorts

Python supports many data types natively, including lists, dictionaries and sets. It supports three kinds of numbers natively: Integers (int), floating point numbers (float) and complex numbers (complex).[2] Integers are simply whole numbers— that is, they are numbers without a decimal component. Floating point numbers have (or at least support) a decimal component. Complex numbers have both a real and an imaginary component, each of which is a float.

On numbers, the usual +, -, *, and / operators are supported, and behave as you would expect. More, the double-star operator, **, is used to denote power, so that 2**8 represents 28. For example,

num = 2**2 + 2.5 print(num)
add.py

prints 6.5, where 2**2 is an int, and both 2.5 and num are float.

Adding to our list of built-in functions, Python implements int(), float(), and complex() to construct their respective objects from strings or literals.

Python can also be used interactively, making for a handy calculator. From your shell, simply enter,

$ python3

to be presented with an interactive Python session. This can be useful for weeding out the details of the language, in addition to doing basic math. (For more serious interactive use, IPython is recommended)

Conditionals

Fundamental to programming, and common to every programming language, is the notion of an expression which resolves to either true or false. For example, 4 > 3 resolves to True. while 4 == 3 resolves to False. (Since = is used to bind values to a variables, Python [and most other programming languages] use == to test whether one object is equivalent to another. Thus, 4 == 3 is read, "Four is equal to three.")

Python uses the bool type (short for Boolean) to represent the the values True and False. Note that these values are capitalized, and are a kind of integer, with False having integer value 0 and True having integer value 1.

The power of programming stems from the ability to test conditions and to act differently in different circumstances. This is called branching, and is rooted in the if statement.[3] In Python, if statements look like this:

num = int(input("What is your number? ")) if num > 100: print("That's a big number!") else: print("Small...")
big-little.py

The else branch is optional, but indentation is significant to Python: Each level of indentation denotes a distinct "block" of code. Blocks of code are run (sequentially) as a whole. Blocks can be of unlimited length, and may themselves contain if statements (and blocks of code...).

  1. The first line of big-little.py does three things. What are they? Explain them in detail.
  2. Try printing 4 < 3. What do you think the result would be?
  3. Is 100 > 100?

While Loops

Loops allow a block of code to be repeated. The block of code to be repeated is referred to as the body of the loop. Python supports two kinds of loop statements: for and while. In a while loop, the block of code is repeated until either the condition evaluates to False, or a break statement is reached.

count = 0 while count < 10: print("Hello there!") count = count + 1
big-little.py

Although the line "count = count + 1" is a mathematical contradiction, in Python, as in most programming languages, the = operator is used to bind a value to a variable, and so is valid code: It simply increments the value of count by one.

It's worth noting here that the body of the while loop executes precisely ten times. On the first run, count has a value of zero; on the last, it has a value of nine. When count is incremented to ten, count < 10 evaluates to False, and the while loop exits. This, as opposed to

count = 1 while count <= 10: print("Hello there!") count = count + 1
big-little2.py

is standard programming practice.

Modules

Alongside the built-in types and functions, additional functions and types can be imported using the import statement. These entities are contained in so-called modules; the most popular module is random, often used in games. Another popular module is NumPy, which offers extensive facilities for scientific computing.

import random num = random.randint(0,100) print("I've got my number!") usr_input = int(input("What is your guess? ")) while usr_input != num: if usr_input < num: print("Too low!") else: print("Too high!") usr_input = int(input("Guess again! ")) print("Correct!")
too-high-too-low.py

In this program, we first import the random module, then use it to generate a random number between zero and 100 (inclusive). Next, we get an initial guess from the user. If that guess is not equal (!=) to the number generated, then the block of code containing the if/else statement is executed. After reporting to the user that the guess is either too high or too low, the user is queried for a new guess, and the new result is compared to the randomly generated number once again.
