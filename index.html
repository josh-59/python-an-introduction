Python

Welcome!

Python is a general-purpose, interpretted programming language. Although widely regarded as appropriate for new programmers, it is not a small language. Moreover, the large ecosystem that has been built around its success can, be a hurdle for the new user. The purpose of this work is to orient the new user to the language.
